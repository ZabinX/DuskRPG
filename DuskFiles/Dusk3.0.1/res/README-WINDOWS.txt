How to Run a Dusk Game on Windows for Dummies


1) Make sure you have downloaded and extracted the DuskServer_DataFiles.zip file.

2) Make that you have downloaded and extracted the DuskServer73_Runnable.zip file into the DuskRPG directory created by the DuskServer_DataFiles.zip file.

3) Execute Dusk.bat

NOTE: To allow other's to access your server require modifications to the conf/prefs file as well.
